** ReadMe_ChildLaborLaws.txt **

This folder contains the data and programs used to generate Table 6.3 in
Mastering Metrics. Table 6.3 2SLS estimates of the returns to schooling where child labor laws are used as an instrument for years of schooling.

Data: 
* /ChildLaborLaws/Data/AA_small.dta

Code: 
* /ChildLaborLaws/Code/AA_regs.do

Output: 
* /ChildLaborLaws/Output/first.xls (first-stage and reduced-form estimates for Panel A of Table 6.3)
* /ChildLaborLaws/Output/second.xls (second-stage estimates for Panel B of Table 6.3)
